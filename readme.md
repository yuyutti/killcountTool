# Kill Count Tool

配信中のキル数をワンクリックで管理＆OBSにリアルタイム表示できるツールです。

👉 FPS・バトロワ配信者におすすめ

キル数以外にも、カウントを配信画面に写す用途などにも活用できます！

## アプリ画面

シンプルで直感的に操作できます。

![アプリケーションUI](./readmeAsset/Screenshot%202026-03-20%20124244.png)

## 主な機能

- OBSにリアルタイム反映
- ワンクリックでキル数を加算 / 減算
- 投げ銭・金額からキル数を自動計算
- 表示テキストを自由に変更可能（例：「キル」→「ビクロイ」）


## こんなときに便利

- キル数カウント配信
- 罰ゲーム系企画（〇キルごとに〇〇）
- 投げ銭連動企画

## OBS連携

アプリで設定したキル数は、自動でテキストファイルに出力されます。

OBSの「テキスト（GDI+）」で読み込むだけで、
配信画面にリアルタイム表示できます。

フォント・色・サイズも自由にカスタマイズ可能です。

OBS側の操作は一度設定すればOK。
以降はアプリ操作だけで自動反映されます。

※以下の2つのファイルが出力されます  
・kill.txt → OBSでそのまま表示（文字付き）  
・killcount.txt → カスタム表示や他ツール連携用（数字のみ）

## 仕組み

アプリ → kill.txt → OBS

アプリで変更した数値が
テキストファイル経由でOBSに反映されます。

## 使い方

1. アプリを起動
2. OBSで「テキスト（GDI+）」を追加
3. 「ファイルから読み込み」をON
4. 「ドキュメント」→「killCountTool」を開く
5. 「kill.txt」を開く

## ファイルの保存場所

kill.txt 及び killcount.txt は以下のフォルダに自動生成されます：

Documents\killCountTool

※ OneDriveを使用している場合は  
OneDrive\Documents\killCountTool になることがあります

## 「キル」の文字を変更する

入力欄の横に表示されている **「キル」という文字をクリック** すると編集できます。  
「ビクロイ」「勝利」など好きな文字に変更してください。

入力欄からフォーカスが外れた時点で保存・反映されます。(何もないところをクリック！)

## 注意

初回起動時にWindowsの警告が表示される場合がありますが、
問題なく使用できます。

「詳細情報」→「実行」をクリックしてください。

![警告1](./readmeAsset/Screenshot%202026-03-20%20125114.png)
「詳細情報」をクリック

![警告2](./readmeAsset/Screenshot%202026-03-20%20124910.png)  
「実行」をクリック  

※一度許可すると、次回以降は通常通り起動します

## 設定リセット

設定をリセットしたい場合は、  
保存先フォルダ内のファイルを削除してください。
